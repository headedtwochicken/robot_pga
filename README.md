# Robot PGA 🤖

An interactive C++ project demonstrating Projective Geometric Algebra (PGA) capabilities. We use PGA math to control a two-joint robot arm and calculate complex kinematics without making the code messy.

## 🎮 Mini-Games
The project includes three interactive mini-games to demonstrate the math in action:
* **Catch Game:** Catch falling objects.
* **Trace Game:** Trace geometric shapes using smooth kinematics.
* **Ping-Pong:** Play a classic game of pong against the robot arm.

## 🛠️ Tech Stack
* **Language:** C++23
* **Graphics:** SFML 3.0.2
* **Testing:** Google Test
* **Build System:** CMake
